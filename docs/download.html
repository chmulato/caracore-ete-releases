Na página do Release você encontra:

  • Minerador40-windows-x64.exe — instalador Windows
  • Minerador40-linux-x86_64.AppImage — pacote Linux
  • Minerador40-macos-universal.dmg — instalador macOS
  • checksums.sha256 e checksums.md5 — validação de integridade
  • INSTALACAO_MULTIPLATAFORMA.md — guia rápido por sistema operacional

Validação SHA-256 por sistema:

  • Windows (PowerShell): Get-FileHash -Path .\Minerador40-windows-x64.exe -Algorithm SHA256
  • Linux: sha256sum Minerador40-linux-x86_64.AppImage
  • macOS: shasum -a 256 Minerador40-macos-universal.dmg

Compare os valores com o arquivo checksums.sha256 publicado no mesmo release.
