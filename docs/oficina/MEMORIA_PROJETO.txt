TESE TÉCNICA ECONÓMICA

ETE aqui significa reutilizar lógica de tanques em série, separação por fases e controlo de precipitação aplicada a refino de concentrados e soluções ricas (PLS), não saneamento doméstico clássico.

Tese de negócio: modularidade tipo ETE pode reduzir CAPEX e permitir escalar por etapas.

COMANDOS CORRIDOS PELA EQUIPA

Minerador 4.0 em desenvolvimento
  python run_minerador40.py
  ou: python -m minerador_40.main

Build de pacote desktop local
  python scripts/build_dist_local.py
  Linux ou macOS: scripts/build_dist_local_unix.sh conforme documentação do script

Streamlit ETE Residencial
  streamlit run src/ete_residencial/app_residencial.py

Streamlit ETE Mineração
  streamlit run src/ete_mineracao/app_mineracao.py

Manual PDF Ouro 4.0
  python gerar_manual_ouro40_pdf.py
  Saída habitual: docs/assets/pdf/Manual_Iniciacao_Ouro40.pdf

Python de referência igual ao CI: 3.10.x


PILARES TÉCNICOS RESUMIDOS

Motor químico com balanço e precipitação por família de metais.
Seletividade por janelas de pH por grupo de lantanídeos.
Cascata industrial em série com métricas de pureza.
Viabilidade com CAPEX, OPEX, payback, VPL, TIR, cenários e sensibilidade reduzida.
Interfaces Streamlit e HTML e camada Minerador para narrativa e relatórios.
