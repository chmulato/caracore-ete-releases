DISTRIBUIÇÃO — REFERÊNCIA PARA A OFICINA

Os guias completos por sistema estão em Markdown na pasta docs/ para utilizadores finais.
Versão do pacote: ver ficheiro VERSION na raiz e tag v correspondente na release.


WINDOWS

Artefacto típico na release: Minerador40-windows-x64.exe (PyInstaller onefile no fluxo de CI).
Runtime WebView2 quando aplicável ao pywebview.
Guia: docs/INSTALACAO_WINDOWS.md


LINUX

Artefacto típico: Minerador40-linux-x86_64.tar.gz — extrair pasta Minerador40, executável junto de _internal.
Dependências de sistema para GTK ou WebKitGTK conforme distro para pywebview.
Guia: docs/INSTALACAO_LINUX.md


MACOS

Artefacto típico: Minerador40-macos-universal.dmg com conteúdo em modo pasta.
Primeira execução pode exigir confirmação no Gatekeeper.
Guia: docs/INSTALACAO_MACOS.md


ÍNDICE E INTEGRIDADE

docs/INSTALACAO_MULTIPLATAFORMA.md — índice geral.
Em cada release: checksums SHA-256 e MD5 junto dos binários.


BUILD LOCAL

scripts/build_dist_local.py — opções documentadas no próprio script e nos guias.
